# ECOWatch

## Setup Grafana

1. Replace folder `public`
   - Source: `grafana/public`
   - Destination: `GrafanaLabs/grafana/public`

2. Tambahkan file `grafana.ini`
   - Source: `grafana/conf/grafana.ini`
   - Destination: `GrafanaLabs/grafana/conf/grafana.ini`
     
3. Replace folder `plugin`
   - Source : `grafana/data/plugins`
   - Destination: `GrafanaLabs/grafana/data/plugins`

4. Replace file `grafana.db`
   - Source : `grafana/data/grafana.db`
   - Destination: `GrafanaLabs/grafana/data/grafana.db`
